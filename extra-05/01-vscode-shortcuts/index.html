Useful shortcuts

Keyboard & mouse useful keys

  1. Home Key
  2. End Key
  3. SHIFT Key + arrows (pgDown, pgUp) - to select text
  4. CTRL + SHIFT Key + arrows (left, right) - to select next word
  5. Mouse dbl click - select a word

Emmet shortcuts

  1. .class-name generates a div with "class-name" class
  2. elem.class-name generates an "elem" element with "class-name" class
  3. elem*n generates n "elements"
  4. parentElem>childElem generates a "parentElement" whith a "childElement" inside
  5. elem1+elem2 generates an "elem1" followed by an "elem2"

VS Code shortcuts

  1. ALT + SHIFT + Down Arrow - duplicates a line (selected block of code)
  2. CTRL + D - selects the next identic word
  3. CTRL + SHIFT + L - selects all the identical words
  4. ALT + Mouse Click - creates a multicursor
  5. ALT + Arrows (up, down) - moves a line (selected lines) up / down
  6. Use editor line arrows to collaps / expand the content of a tag
